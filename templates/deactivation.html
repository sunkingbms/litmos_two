Deactivate Users

Instructions

Upload a CSV file containing usernames of Litmos users you want to deactivate. The system will process each user and attempt to deactivate them in Litmos.

The CSV file must include a column named username with the Litmos usernames.
Required: Your CSV file must contain between 30 and 100 users per upload.
This operation will set the user's status to "Inactive" in Litmos. Deactivated users won't be able to log in to the Litmos system.

Upload CSV File

File must be in CSV format with a "username" column.
